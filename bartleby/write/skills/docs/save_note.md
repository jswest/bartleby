Save a research note for later reference.

Use this to record synthesized conclusions, key facts, or important findings that you want to reference in future questions. Notes are saved as Markdown files in the findings directory.

**When to use:**
- You've synthesized an important conclusion from multiple sources
- You've found a key fact worth remembering
- You want to build up findings across multiple questions

**Do NOT save:**
- Restatements of the user's question
- Raw search results
- Trivial observations
